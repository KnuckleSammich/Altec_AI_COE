Overview

A program that analyzes large batches of part print PDFs and checks for common design errors and manufacturing difficulties. The current build identifies bend dimensions and compares them to Plant 14's materials list to flag minimum flange length issues before parts reach the shop floor.

Key Capabilities

  • Batch processing of part print PDFs
  • Detects title block zones and staged dimension extraction
  • Identifies bend callouts and flange dimensions
  • Cross-references Plant 14 materials list for minimum flange length checks
  • Debug overlays and GUI for iterative validation

Impact

Catches preventable manufacturing issues at the design-review stage, reducing downstream rework, scrap, and production delays. Provides a scalable way to enforce design-for-manufacturing rules across thousands of prints.

Tools Used

  • Python (PDF parsing, image processing)
  • GUI framework for desktop interface
  • SolidWorks PDF outputs
